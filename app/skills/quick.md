# Quick Answer

icon: flash_on

> Trả lời nhanh câu hỏi đơn giản

## Usage

Sử dụng lệnh `/quick` cho câu hỏi cần trả lời nhanh.

## Prompt

Trả lời NGAY LẬP TỨC câu hỏi sau một cách ngắn gọn nhất có thể. Không giải thích dài dòng. Chỉ đưa ra câu trả lời trực tiếp.

Nếu là câu hỏi yes/no → Trả lời Yes/No
Nếu là câu hỏi số → Trả lời số
Nếu là câu hỏi định nghĩa → 1-2 câu
Nếu là câu hỏi code → Code snippet ngắn nhất
