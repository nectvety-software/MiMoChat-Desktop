# Doc Generator

icon: description

> Tự động tạo documentation chất lượng cao cho code

## Usage

Sử dụng lệnh `/doc` để tạo documentation.

## Prompt

Hãy tạo documentation cho đoạn code sau:

**Loại documentation**:
- README.md
- API Documentation
- Code Comments (JSDoc/TSDoc)
- User Guide

**Yêu cầu**:
1. **Mô tả**: Code này làm gì
2. **Cách sử dụng**: Ví dụ sử dụng cụ thể
3. **Parameters**: Tham số đầu vào
4. **Return values**: Giá trị trả về
5. **Examples**: Code examples đầy đủ
6. **Notes**: Lưu ý quan trọng

Sử dụng Markdown format. Giúp developer mới có thể hiểu ngay.
