# API Builder

icon: api

> Thiết kế và tạo REST API endpoint hoàn chỉnh

## Usage

Sử dụng lệnh `/api` để tạo API endpoint.

## Prompt

Hãy thiết kế REST API endpoint:

**Yêu cầu**:
- Method: [GET/POST/PUT/DELETE]
- Path: [endpoint path]
- Mục đích: [mô tả ngắn]

Hãy tạo:
1. **API Design**: OpenAPI/Swagger style description
2. **Request**: Body, params, headers
3. **Response**: JSON response mẫu (success + error)
4. **Validation**: Input validation rules
5. **Code**: Implementation code (Node.js/Python/etc.)
6. **Error handling**: Các lỗi có thể xảy ra
7. **Test**: API test examples

Sử dụng RESTful conventions và best practices.
