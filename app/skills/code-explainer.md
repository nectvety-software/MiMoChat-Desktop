# Code Explainer

icon: description

> Giải thích code một cách dễ hiểu, phù hợp cho người mới

## Usage

Sử dụng lệnh `/explain` để giải thích code.

## Prompt

Hãy giải thích đoạn code sau một cách đơn giản và dễ hiểu:

1. **Tổng quan**: Code này làm gì? (1-2 câu)
2. **Cách hoạt động**: Giải thích từng phần
3. **Flow**: Mô tả flow thực thi (có thể dùng diagram text)
4. **Ví dụ**: Cho ví dụ cụ thể nếu có thể
5. **Thuật ngữ**: Giải thích các thuật ngữ quan trọng

Giải thích như đang dạy cho người mới học lập trình.
Sử dụng analogies đời thường nếu phù hợp.
