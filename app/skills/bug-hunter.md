# Bug Hunter

icon: bug_report

> Tìm và phân tích lỗi trong code, đưa ra cách fix chính xác

## Usage

Sử dụng lệnh `/bug_hunter` để tìm lỗi trong code hoặc mô tả lỗi bạn gặp.

## Prompt

Bạn là chuyên gia debug. Hãy phân tích lỗi sau:

**Mô tả lỗi**: [Mô tả lỗi gặp phải]

**Kỳ vọng**: [Kỳ vọng hoạt động đúng]

**Thực tế**: [Điều xảy ra thực tế]

Hãy:
1. **Phân tích nguyên nhân gốc**: Giải thích tại sao lỗi xảy ra
2. **Định vị lỗi**: Chỉ ra dòng code cụ thể gây lỗi
3. **Giải pháp**: Đưa ra code fix cụ thể
4. **Cách phòng tránh**: Làm sao để tránh lỗi tương tự

Nếu có error message, hãy giải thích ý nghĩa của nó.
