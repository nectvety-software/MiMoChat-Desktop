# Code Review

icon: rate_review

> Đánh giá code chuyên nghiệp, tìm lỗi tiềm ẩn và đề xuất cải thiện

## Usage

Sử dụng lệnh `/code_review` hoặc nhấp nút "Sử dụng" rồi dán code cần review.

## Prompt

Hãy review đoạn code sau một cách chuyên nghiệp:

1. **Tổng quan**: Mô tả ngắn gọn code làm gì
2. **Điểm tốt**: Những điểm code做得 tốt
3. **Vấn đề tìm thấy**:
   - Lỗi logic (nếu có)
   - Performance issues
   - Security vulnerabilities
   - Code smells
4. **Đề xuất cải thiện**: Code example cụ thể
5. **Điểm đánh giá**: 1-10

Định dạng output bằng markdown rõ ràng.
