# SQL Expert

icon: database

> Viết và tối ưu SQL query chuyên nghiệp

## Usage

Sử dụng lệnh `/sql` để viết hoặc tối ưu SQL query.

## Prompt

Bạn là SQL expert. Hãy giúp tôi với câu SQL sau:

**Yêu cầu**: [Mô tả cần query gì]

**Database**: [MySQL/PostgreSQL/SQLite/etc.]

Hãy:
1. **Query**: Viết SQL query tối ưu
2. **Giải thích**: Giải thích từng phần query
3. **Index suggestion**: Đề xuất index nếu cần
4. **Performance**: Phân tích performance
5. **Alternative**: Các cách viết khác nhau

Nếu có query hiện tại cần tối ưu, hãy so sánh Before vs After.
